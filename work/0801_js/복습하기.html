ex7 : 증감연산자: 처리순서 매우 중요

증감연산자의 종류는 ++(1씩 더하기) --(1씩 빼기) 뿐입니다.
다만! 변수 앞에 올 땐 누구보다 빠르게 처리하지만, 변수 뒤에선 행이 바뀔 때 수행됩니다.
a=3일 때,
println(++a); // 4, a = 4
println(a++); // 3, a = 4

ex8 : 논리연산자

논리연산자의 종류는   &&(and)   ||(or)  !(Not)  3가지가 있다. 논리연산자는 boolean타입 자료형이므로 true 혹은 false 값을 갖는다.
따라서 선행 조건의 결과에 따라 후속 조건을 수행하지 않을 수 있다.
&&의 경우, 선행조건의 값이 false라면 어값 false이므로 STOP.
||의 경우, 선행조건의 값이 true라면 어값 true 이므로 STOP.
